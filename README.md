# Meme Generator

Created using React v6 and <https://memegen.link/> meme generator.

## Installation

Ensure you have npm or yarn installed in your system

```bash
git clone https://github.com/Mugambi645/meme-generator
```

## Usage

```
cd meme-generator
npm install or yarn install
npm start
```
